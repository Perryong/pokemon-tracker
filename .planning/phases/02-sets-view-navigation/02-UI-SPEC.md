| Token | Value | Usage |

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFFFF` / `hsl(var(--background))` | App background and main page surfaces |
| Secondary (30%) | `#F5F5F5` / `hsl(var(--muted))` | Card media containers, muted panels, filter empty/loading surfaces |
| Accent (10%) | `#171717` / `hsl(var(--primary))` | Active nav state, focused filter control ring/text emphasis, primary interactive emphasis only |
| Destructive | `#EF4444` / `hsl(var(--destructive))` | Error surfaces and retry emphasis only |

Accent reserved for: active view indicator (`sets`/`collection`), selected/focused filter controls, and explicit primary action emphasis.  
Do **not** use accent for all badges; completion badge remains semantic success styling (emerald) to preserve ŌĆ£100% completeŌĆØ scanability.
