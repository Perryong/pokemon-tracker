## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline icon/text micro-gaps inside badges and compact labels |
| sm | 8px | Tight control spacing (filter rows, compact actions) |
| md | 16px | Default spacing for card content, control groups, and body layout |
| lg | 24px | Section spacing between album header, controls, and grid |
| xl | 32px | Larger layout separation at tablet/desktop widths |
| 2xl | 48px | Major section break before/after album content blocks |
| 3xl | 64px | Page-level breathing room on wide layouts |

Exceptions: 44px minimum touch target for icon-only controls (back button and icon-only actions).

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 400 | 1.5 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

Rules:
- Exactly two font weights in this phase: 400 and 600.
- Set title in album header uses Display.
- Card names, section titles, and state headings use Heading.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFFFF` / `hsl(var(--background))` | Primary app/page surface and album canvas |
| Secondary (30%) | `#F5F5F5` / `hsl(var(--muted))` | Filter panels, muted containers, skeleton/empty-state surfaces |
| Accent (10%) | `#171717` / `hsl(var(--primary))` | Active album controls (size/filter/search focus), key primary emphasis |
| Destructive | `#EF4444` / `hsl(var(--destructive))` | Remove/error actions and retry error emphasis only |

Accent reserved for: active size mode, active ownership filter, focused search input ring, and primary control emphasis in album header.

Owned-state semantic styling:
- Owned cards MUST use explicit success styling (`ring-green-500` + checkmark indicator) and must not be replaced with neutral accent color.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | **Mark as Owned** |
| Empty state heading | No Cards Found |
| Empty state body | No cards match your current filters. Clear Filters to see all cards in this set. |
| Error state | Error Loading Cards. Select **Retry** to try again. If the issue persists, return to Sets and reopen the album. |
| Destructive confirmation | Remove from Collection: no modal; immediate action with destructive styling + toast feedback, reversible by marking owned again |

Additional locked labels:
- Size toggle options: `Small`, `Medium`
- Ownership filter options: `All`, `Owned`, `Missing`
- Search placeholder: `Search cards...`
- Stats footer labels: `Owned`, `Missing`, `Completion`
