2. **useSetCompletion hook**:
   ```typescript
   export function useSetCompletion(
     setCardIds: string[],
     ownedCards: Record<string, boolean>
   ): CompletionStats {
     return useMemo(() => {
       const total = setCardIds.length
       if (total === 0) {
         return { owned: 0, missing: 0, total: 0, percentage: 0 }
       }
       
       const owned = setCardIds.filter(id => ownedCards[id]).length
       const missing = total - owned
       const percentage = (owned / total) * 100
       
       return { owned, missing, total, percentage }
     }, [setCardIds, ownedCards])
   }
   ```

3. **useMemo import** — ensure `useMemo` is imported from React at top of file

The hook:
- Takes `setCardIds` (array of card IDs in a set) and `ownedCards` (from useCollection)
- Returns computed stats: owned count, missing count, total count, completion percentage
- Uses useMemo for performance — only recomputes when inputs change
- Is deterministic and recomputable from source data (not persisted)
