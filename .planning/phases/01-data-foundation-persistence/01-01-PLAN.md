{
  id: string           // "base1", "swsh1"

  symbol?: string      // URL to set symbol

  cards: CardResume[]  // Array of card objects
}

// SerieResume (from tcgdex.fetch('series'))
{
  id: string       // "base", "sword-shield"
  name: string     // "Base", "Sword & Shield"
  logo?: string    // URL to series logo
}

// Card
{
  id: string         // Global ID like "swsh1-1"
  localId: string    // Number within set like "1"

  image?: string     // Single URL (NOT images.small/large)
  rarity?: string
  // ... other fields
}

3. **Create adapter functions** (TCGdex → App types):

   `normalizeTCGSet(tcgSet: SetResume | Set): PokemonSet`:
   - Map `cardCount.official` → `printedTotal`
   - Map `cardCount.total` → `total`
   - Map `logo` → `images.logo`
   - Map `symbol` → `images.symbol`
   - For `series`: use `tcgSet.serie?.name ?? 'Unknown'` (only full Set has serie)
   - For `legalities`: map `legal.standard` boolean → `'legal'` string, same for expanded
   - Set `releaseDate` and `updatedAt` from Set object or empty string for SetResume

   `normalizeTCGCard(tcgCard: Card | CardResume, setData?: Set): PokemonCard`:
   - Map `localId` → `number`
   - Map `image` (single URL) → `images: { small: image, large: image }` (use same URL for both)
   - Set `supertype` to `'Pokémon'` as default if not present
   - Set `subtypes` to `[]` if not present
   - Build `set` property from setData if provided

   `normalizeTCGSeries(tcgSeries: SerieResume): Series`:
   - Direct mapping of id, name, logo

4. **Export all types and adapters**.
