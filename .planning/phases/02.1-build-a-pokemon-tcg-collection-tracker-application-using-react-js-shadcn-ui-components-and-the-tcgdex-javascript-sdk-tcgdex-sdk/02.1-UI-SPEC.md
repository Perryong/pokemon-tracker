## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label micro gaps |
| sm | 8px | Badge clusters, compact control spacing |
| md | 16px | Default control spacing and card body padding |
| lg | 24px | Section padding and stack separation |
| xl | 32px | Major layout gaps in page sections |
| 2xl | 48px | Large section breaks |
| 3xl | 64px | Page-level vertical rhythm |

Exceptions (standard-scale only): minimum interactive target uses 48px; fixed stats/footer bars use 8px or 16px vertical padding depending on density needs.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 400 | 1.4 |
| Heading | 20px | 600 | 1.2 |

Hard limit for this phase: use only these 4 sizes and 2 weights to reduce variance during hardening.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(var(--background))` (light: `0 0% 100%`) | Page background and main surfaces |
| Secondary (30%) | `hsl(var(--muted))` / `hsl(var(--card))` (light: `0 0% 96.1%` / `0 0% 100%`) | Cards, filter panels, skeleton containers, sticky/fixed bars |
| Accent (10%) | `hsl(var(--primary))` + semantic completion accents (emerald/green) | Primary CTA emphasis, active selection state, completion/owned indicators |
| Destructive | `hsl(var(--destructive))` (light: `0 84.2% 60.2%`) | Remove-from-collection and destructive error actions only |

Accent reserved for: selected pagination item, active filter value emphasis, owned/completion indicators (`✓ Complete`, owned ring/badges), and primary confirm actions. Do not apply accent styling to all controls.

---

## Visual Hierarchy

- **Primary focal point:** set/card content grid and card ownership states.
- **Secondary focal point:** filter/search/size controls that change visible results.
- **Tertiary focal point:** sticky/fixed status surfaces (progress/stats footer).
- Icon-only controls must include accessible label text via `aria-label` and/or adjacent text label.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Retry Loading |
| Empty state heading | No Sets Found / No Cards Found |
| Empty state body | No Pokémon card sets or cards match your current filters. Clear filters to continue browsing. |
| Error state | Error Loading Sets / Error Loading Cards. Show API error message and provide Retry Loading action. |
| Destructive confirmation | Remove from Collection: no modal confirmation; execute immediately with destructive styling and toast feedback; action is reversible via Add to Collection |

Hardening constraint: preserve existing v1 wording unless text is ambiguous during smoke validation.
